## 📋 How It Works

1. Select Job Role
2. Select Experience Level
3. Select Education Qualification
4. Select Location
5. Choose Your Skills
6. Click Predict Salary

The application generates:

- Predicted Salary Range
- Employability Score
- Job Readiness Rating
- Skills Overview
- Improvement Suggestions
